RESUMO DO PROJETO - PM AI MVP
Ferramenta de Gestão de Projetos com Inteligência Artificial

================================================================================
CONTEXTO DO PROJETO
================================================================================

VISÃO GERAL:
O PM AI MVP é uma plataforma SaaS interna que centraliza a gestão de implantação de projetos, 
substitui planilhas manuais e automatiza a captura de ações a partir de reuniões, formando 
uma base de conhecimento (RAG) para consultas inteligentes.

PROBLEMA RESOLVIDO:
- Gestão manual de projetos via planilhas (ineficiente e propenso a erros)
- Falta de centralização de informações de projetos
- Ausência de automação para captura de ações de reuniões
- Dificuldade em manter histórico e base de conhecimento dos projetos

SOLUÇÃO PROPOSTA:
Plataforma web moderna com FastAPI (backend), React (frontend) e PostgreSQL com pgvector 
para IA, oferecendo gestão completa de projetos, checklists, ações e futuramente 
visualizações avançadas e análise preditiva.

================================================================================
REQUISITOS FUNCIONAIS (MVP)
================================================================================

1. AUTENTICAÇÃO E USUÁRIOS
   - Login via Google (integração OAuth)
   - Gestão de perfis de usuário
   - Controle de acesso por projeto

2. GESTÃO DE PROJETOS
   - Criação e edição de projetos
   - Metadados: portfolio, vertical, produto, descrição
   - Listagem e busca de projetos
   - Status e acompanhamento

3. CHECKLISTS E GRUPOS
   - Criação de grupos de checklist por projeto
   - Itens tipificados: Ação ou Documentação
   - Atualização de status dos itens
   - Organização hierárquica

4. CENTRAL DE AÇÕES
   - Criação de itens de ação
   - Tipos: Ação Pontual, Pendência, Chamado, Bug
   - Filtros por tipo e status
   - Atribuição e prazos

5. INTERFACE INTUITIVA
   - Dashboard principal com visão geral
   - Navegação clara entre módulos
   - Design responsivo (mobile-first)
   - Feedback visual para ações do usuário

TECNOLOGIAS E ARQUITETURA
================================================================================

BACKEND:
- FastAPI (Python) - API moderna e rápida
- PostgreSQL 16 + pgvector - Banco relacional + vetores para IA
- SQLAlchemy 2.0 - ORM assíncrono
- JWT para autenticação
- Docker para containerização

FRONTEND:
- React 18 + Vite - Interface moderna e responsiva
- React Router - Navegação SPA
- Axios - Integração com APIs
- CSS moderno com flexbox/grid

INFRAESTRUTURA:
- Docker Compose para desenvolvimento
- PostgreSQL com extensão pgvector
- Redis (futuro) para cache
- n8n (futuro) para automações
